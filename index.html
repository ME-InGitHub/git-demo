推送到远程仓库的三种方式：

完整版：

git push 库地址 分支名称

简写版1：

1. 先取别名：git remote add 别名 库地址

2. 使用方式：git push 别名 分支名称

简写版2：

1. 让git记住仓库名称：git push -u 库地址 分支名称

2. 使用方式：git push
